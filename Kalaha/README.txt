Run kalaha.py followed by 1, 2, or 3 depending if you want to play against Monte Carlo, Minimax or make the AIs play against each other.
	1: Monte Carlo
	2: Minimax
	3: Monte Carlo vs Minimax. Monte Carlo plays first
	
Example command:
	$ python kalaha.py 2

From there on, add algorithm parameters as requested and follow the instructions on how to play that appear on the screen
